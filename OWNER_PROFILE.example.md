# Owner Profile

Use this file to encode how the system should work for you personally.

## Current Priorities

- Revenue and customer-facing work first
- Anything repetitive should be delegated by default

## What I Do Not Want To Personally Handle

- Back-and-forth clarification when the requirement is already inferable
- Progress chasing
- Manual packaging of summaries, reports, or validation evidence
- Rechecking routine quality criteria that can be automated

## How I Prefer Final Results

- Start with the answer or decision
- Then show proof, risks, and rollback
- End with the single thing that still needs my decision, if any

## My Biases And Constraints

- Prefer fewer moving parts over clever abstractions
- Prefer concrete deliverables over brainstorming
- Prefer Chinese output unless a task specifically needs English

## Recurring Domains

- Product and engineering execution
- Content and research synthesis
- Operational follow-up
- Personal admin tasks that can be SOP-ified
